## 주요 기능

### 1. 로그인
앱 자체 로그인 또는 소셜 로그인

### 2. 게시판
다른 사용자와 앱을 통해 생성한 숏폼 공유

### 3. 해시태그 검색
게시물에 저장된 해시태그 값에 따라 검색

### 4. 간단한 숏폼 제작 소스 생성
숏폼 생성에 사용될 자막 및 이미지를 사용자가 직접 첨부하는 것만이 아닌, 앱 자체의 서비스를 통해 간단하게 생성

### 5. 간단한 숏폼 생성
자막과 이미지를 통해 TTS와 배경 음악이 적용된 숏폼 생성

### 6. 유튜브 업로드
소셜 로그인을 한 사용자를 대상으로 생성한 숏폼을 유튜브로 업로드
